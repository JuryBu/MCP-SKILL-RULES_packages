Portable Codex 工具包测试页

用于验证 web-fetcher 的正文提取、截图和摘要能力。

背景

这个页面没有外部资源，适合在离线或半离线环境里做基础 smoke test。

测试内容

页面包含标题、正文、提示块和足够的垂直高度，用于确认正文抽取和截图是否稳定。

如果工具能读到这句话，说明本地 HTML 读取路径基本正常。

长页面段落

第一段：Codex 通过 MCP 调用工具时，应返回可理解、可追踪的结果。

第二段：本地工具包默认把数据放在用户目录下的独立数据根，避免污染安装目录。

第三段：测试结果不要求完全一致，但应能证明 endpoint、工具注册和基本 I/O 工作正常。
